RKANT — Aparato crítico del Nuevo Testamento

Elige un libro, capítulo y versículo en la barra superior, o escribe una referencia en el buscador (por ejemplo Mt 1:1, gal 1 6, apoc 22:21).

  • Cada versículo se carga con el texto griego, el aparato crítico, la tabla de manuscritos y las referencias cruzadas.
  • Pasa el cursor sobre cada sigla para ver el manuscrito completo.
  • Usa las flechas ← / → o los botones Anterior / Siguiente para leer secuencialmente.
